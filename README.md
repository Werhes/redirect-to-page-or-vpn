# 🔀 Redirect to Page or VPN

Простой, быстрый и удобный инструмент для автоматического перенаправления пользователей на заданную веб-страницу или прямую ссылку для подключения к VPN (VLess, OpenVPN и др.).

## 📌 Особенности
* Мгновенный редирект без лишних задержек.
* Гибкая настройка целевых ссылок.
* Удобная конфигурация названия и параметров.
* Легкий деплой на любой сервер.

---

## ⚙️ Как настроить название и параметры

Все основные настройки подписки (включая отображаемое название и ссылки для перенаправления) выполняются через саму же ссылку
   # 🏳️‍🌈 Ссылка на подписку для VPN-клиента - пример
   [*https://werhes.github.io/redirect-to-page-or-vpn/?redirect_to=ВАШЕ_ПРИЛОЖЕНИЕ_ДЛЯ_VPN://install-sub?url=ВАША_ССЫЛКА_НА_ПОДПИСКУ&name=ВАШЕ_ИМЯ_ПОДПИСКИ*]

   # 😉 Как настроить:

   • Как настроить приложение, в котором и будет ваша подписки на VPN:
   ***Найдите строку ?reditect_to= И после равно впишите название приложения - например: ?redirect_to=happ
